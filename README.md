# Flutter Calculator
Calculadora desenvolvida em flutter capaz de realizar operações de soma, subtração, divisão e multiplicação

## Como rodar
### 1. Clonar o repo
```bash
git clone https://github.com/KenzoUMZ/flutter-calculator
```

### 2. Instalar as dependências
```bash
flutter pub get
```

### 3. Rodar
```bash
flutter run
```

## Testes
### 1. Rodando os testes
```bash
flutter test
```
